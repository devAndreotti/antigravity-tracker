# Antigravity Tracker Dev Workflow

// turbo-all

## Run in development mode

```bash
cd d:\Dev\Projects\Antigravity\antigravity-tracker && npm run dev
```

## Build for production

```bash
cd d:\Dev\Projects\Antigravity\antigravity-tracker && npm run build
```

## Build Windows installer

```bash
cd d:\Dev\Projects\Antigravity\antigravity-tracker && npm run build:win
```
